Политика конфиденциальности виджета Nexus Looker

Редакция от {{POLICY_DATE}}

Настоящая политика описывает, какие данные обрабатывает виджет Nexus Looker (далее — «Виджет») для amoCRM и что с ними происходит. Виджет разработан и поддерживается {{COMPANY_NAME}} (далее — «Разработчик»).

1. Что делает Виджет

Виджет добавляет в интерфейс amoCRM кнопку предпросмотра рядом с файлами-вложениями и показывает содержимое файла в модальном окне, не требуя скачивания.

2. Какие данные обрабатываются

  • Содержимое файлов-вложений — только тех, для которых пользователь явно нажал кнопку предпросмотра. Виджет никогда не обрабатывает файлы фоном.
  • Технические данные запросов к серверу Разработчика (IP-адрес, адрес кабинета amoCRM в заголовке Origin) — в минимальном объёме, необходимом для защиты сервиса от злоупотреблений (ограничение частоты запросов).

Виджет не собирает и не хранит персональные данные клиентов amoCRM, не читает сделки, контакты, переписку и любые данные CRM, кроме файла, открытого пользователем.

3. Куда уходят файлы при предпросмотре

  • PDF, изображения (включая SVG), текст, Markdown — обрабатываются целиком в браузере пользователя. Никуда не передаются.
  • Устаревшие форматы Office (.doc, .xls, .ppt, .rtf, .odt) — передаются на сервер Разработчика, конвертируются в PDF и сразу возвращаются в браузер. Файл не сохраняется на сервере; содержимое и имена файлов не логируются.
  • Современные форматы Office (.docx, .pptx, .xlsx, .csv) — для отображения используется Microsoft Office Online Viewer. Файл передаётся на сервер Разработчика и временно (не более 5 минут) публикуется по непредсказуемому одноразовому адресу, с которого его скачивает сервис Microsoft. По истечении срока файл автоматически удаляется. Обработка файла сервисом Microsoft регулируется условиями Microsoft (Microsoft Privacy Statement).

4. Хранение и логирование

Сервер Разработчика не хранит файлы дольше времени, необходимого для конвертации или предпросмотра (максимум 5 минут для Office-форматов). Содержимое файлов не логируется. Технические журналы сервера ограничены данными, необходимыми для обеспечения работоспособности и защиты от злоупотреблений: служебный идентификатор запроса, тип (расширение) и размер файла, длительности этапов обработки, код результата и страна запроса. Имена файлов, их содержимое и IP-адреса в журналы не записываются; журналы автоматически удаляются по ротации (порядка 30 дней).

5. Передача третьим лицам

Единственное третье лицо, получающее содержимое файлов, — Microsoft (только для форматов .docx, .pptx, .xlsx, .csv, см. раздел 3). Данные не продаются и не передаются иным лицам.
