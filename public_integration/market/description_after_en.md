# Description AFTER install (English marketplace / Kommo)

> Shown to accounts that installed the integration.

---

**Nexus Looker is installed — how to use it**

1. Open a lead, contact or company card.
2. A round blue eye button now appears next to every supported file in the notes feed.
3. Click it — the file opens in a preview window. Close it with the cross button,
   a click outside the window, or the Esc key. The Download link in the header is
   there if you still need the file on disk.

**Supported formats:** PDF; images (JPG, PNG, GIF, WebP, SVG); DOCX, XLSX, PPTX, CSV;
legacy DOC, XLS, PPT, RTF, ODT; plain text and Markdown.

**Size limits:** Office formats — up to 15 MB; legacy formats — up to 50 MB;
Markdown — up to 2 MB. Larger files can be downloaded as usual.

**No configuration needed.** The integration works right after install. You can
disable it at any time in the integrations settings.

**Privacy:** PDF, images and text never leave your browser. When previewing
.docx/.pptx/.xlsx/.csv files they are temporarily (up to 5 minutes) processed by
Microsoft Office Viewer — see the
<a href="https://nexus-oko.naithon.one/legal/privacy_en.html">privacy policy</a>.

email {{SUPPORT_EMAIL}}. Note: support is provided by the integration developer.
