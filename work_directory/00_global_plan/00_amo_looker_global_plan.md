## Ключевые зависимости и риски

1. **T12 (доступы) — внешний блокер E2 и T19**, теперь в E1S1 с запросом в день старта. Контингенси: при задержке E3S1 выполняется до E2; релиз v1.0 возможен без legacy-форматов (конвертер догоняет отдельным релизом).
2. **Docker локально отсутствует** (проверено) — конвертер разрабатывается на dev-боксе после T12 либо ставится Docker Desktop.
3. **T8 (PDF)** — главная техническая неопределённость E1; средний вариант A2 (blob→iframe) снимает риск дешевле pdf.js.
4. **Токен конвертера извлекаем из клиентского виджета** — принятая модель угроз: токен отсекает интернет-шум, реальная защита — Origin-allowlist + лимиты + изоляция контейнера (T14).
5. DOM amoCRM может измениться — селекторы зафиксированы разведкой 10.06.2026; kill-switch (T17) даёт быстрый откат на проде; тесты T21 ловят регресс на фикстурах.
6. QA писем/чата может быть недоступен в dev-кабинете — честный статус по типам лент в T5, добор на прод-прогоне T20.
