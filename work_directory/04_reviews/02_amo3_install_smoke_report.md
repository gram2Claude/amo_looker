## Грабли установки (для повторения процедуры)

- ❌ Путь «amoМаркет → Мое приложение» — НЕ деплоит архив на CDN без модерации маркетплейса (интеграция `toolkeeperlooker` создана этим путём — дубликат, виджет «пустой»; можно отключить).

- Каждый ресабмит формы создания = новый код подтверждения (старый сгорает).

- amo массово кеширует: ошибки валидации остаются в DOM после исправления — проверять после reload.
