# Букмарклет для ручного теста Nexus Looker на техническом аккаунте

**Назначение:** активировать виджет на странице карточки в кабинете venskons78 (технический аккаунт, где лоадер amoCRM сам виджеты не грузит). Работает в ЛЮБОМ браузере, где ты залогинен в amoCRM — обычный Chrome подойдёт.

## Букмарклет (строка закладки)

```
javascript:(function(){var s=document.createElement('script');s.src='https://nexus-oko.naithon.one/widget/boot.js?'+Date.now();document.body.appendChild(s);})();void(0);
```

## Как пользоваться

1. **Создай закладку** в браузере: ПКМ по панели закладок → «Добавить страницу» → Имя: `Nexus Looker`, URL: вставь строку выше.
2. **Залогинься** в `https://venskons78.amocrm.ru` (обычный твой Chrome).
3. **Открой карточку сделки** с прикреплёнными файлами (например сделку 3177663).
4. **Нажми закладку** `Nexus Looker` — у файлов в ленте появятся синие «глазики».
5. Клик по глазику → модалка с предпросмотром (PDF/DOCX/XLSX/изображение/текст; .doc/.xls идут через конвертер).
6. Перезагрузил страницу — нажми закладку снова (активатор не персистентный, это нормально для теста).

## Что под капотом
- Закладка грузит `boot.js` с нашего сервера `nexus-oko.naithon.one/widget/`.
- `boot.js` перехватывает `define`, поднимает виджет (`render`+`init`), vendor (docx-preview/jszip/xlsx) тянет с того же сервера.
- Токен конвертера зашит в `boot.js` НА СЕРВЕРЕ (не в этой закладке) — поэтому сама строка закладки безопасна. Сервер за CORS-allowlist (venskons78/toolkeeper).

## Статус проверки
- Хостинг (`/widget/boot.js`, `script.js`, `style.css`, `vendor/*`) — отдаётся по HTTPS 200 ✅.
- boot инстанцирует виджет (`__nexusBooted` ставится, стиль подхватывается) ✅.
- Финальное «глазики на залогиненной карточке через букмарклет» — подтверждается тем же путём, что отлажен в QA (инъекция бандла → 6/6 глазиков, рендеринг реальных docx/xlsx); на момент сборки сессия в тестовом браузере слетела, проверь у себя.

## Это тест-инструмент, не прод
Букмарклет — для ручного дог-фудинга на техническом аккаунте. На обычном рабочем аккаунте виджет ставится штатно (приватная интеграция) и грузится сам, без закладки.
